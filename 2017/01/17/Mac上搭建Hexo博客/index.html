Mac上搭建Hexo博客

发表于 2017-01-17

导语

为搭建Hexo博客,踩了无数的坑,终究是搭建成功了,高兴之余记录一下.

之前买过香港主机,用wp建站,试过之后每隔个1,2月时不时会挂掉,咨询空间商告诉说主机IP受到大规模DDOS攻击,导致不能访问,问题往复几次,个人接受不了,遂放弃.转向Github, 使用Github搭建博客也有好处,最直接的一点就是免费.话不多说,开始记录.

Mac上搭建基于GitHub的Hexo博客可分5步:

  1. 搭建github仓库及域名解析;
  2. Hexo关联Github;
  3. 安装主题theme;
  4. 绑定个人域名;
  5. 创建并发布博客文章;

一 搭建github仓库及域名解析

  1. 创建github账号(github官网),一定要验证邮箱(验证邮箱)
    (这是作者新注册的github账号 用户名:ceshiblog 密码:y**95**84 邮箱:134****978@qq.com )
  2. 登录github,右上角点+按钮,创建仓库New repository
  3. 进入Create a new repository页面,
    • ①输入Repository name,要注意格式:你的github用户名.github.io,比如:ceshiblog.github.io
    • ②输入Description描述,比如个人博客,对于描述随意填,是可以改的
    • ③选择public
    • ④点击 Create repository
  4. 进入仓库页面 (作者的仓库页面) ,点击setting
  5. 向下滚动,设置主题(Change theme),选择喜欢的主题,随意选个就行

    选择一个主题并确定
  6. 仓库建好了,博客网站也基本建好 (https://ceshiblog.github.io)

  7. 绑定个人域名(关于域名的购买,自行百度,其实非常简单)

    • 现在我们不想通过输入https://ceshiblog.github.io来访问个人博客网站,而是希望通过自己的域名(如:http://ceshiblog.xyz)来访问

    • 下面详细步骤

      • ① 进入仓库页面的设置(Settings)选项卡(作者的仓库页面 )
      • ② 往下滑动,找到域名卡片(Custom domain),输入购买的域名(ceshiblog.xyz是作者在阿里云买的域名,约¥1元/年),然后保存
      • ③ 保存成功的显示: Custom domain “ceshiblog.xyz” saved

      • ④ 到域名商那里去域名解析(域名在哪里买的就到哪里去解析,作者去的是阿里云)

        名词解释:
    域名解析: 通过域名查找IP地址并访问空间的过程叫做域名解析.  
     CNAME: 如果将域名指向一个域名，实现与被指向域名相同的访问效果，需要增加CNAME记录.  
        @ : 将域名解析为example.com（不带www），填写@或者不填写；  
      www : 将域名解析为www.example.com，填写www；

      • ⑤ 等待10分钟,就可以通过个人域名(http://www.ceshiblog.xyz 或http://ceshiblog.xyz)访问博客网站了
  8. 手动clone一份代码,因为后面或许要用到(这一步可不做)
    打开仓库页面(作者的仓库页面 ),
    点击右侧绿底Clone or download,点击download zip,将代码clone到本地电脑,保存即可.
  9. 建站整理
    仓库页面 : https://github.com/ceshiblog/ceshiblog.github.io 账号密码:github的登录账号
    个人博客: http://ceshiblog.xyz

建站到这里基本完成1/5了,剩下的就是博客的内容创建发布了,对于新手来说发布博客的操作越简单越好.
作者觉得使用hexo写博客相对简单,请往下看

二 Hexo关联Github

上面成功搭建Github仓库, 这一步主要是在本地Mac搭建Hexo环境并关联Github.

Hexo 是一个快速、简洁且高效的博客框架. Hexo官网文档
安装Hexo必须有两个先决条件

  1. Node.js
  2. git
  1. 下载Node.js和git并安装















    软件 下载地址 解释
    ①下载Node.js https://nodejs.org/en/ 下载v6.9.2 LTS 一路安装即可
    ②下载git https://git-scm.com 或到Hexo官网文档下载

  2. 安装Hexo

















    步骤 命令 解释
    ① $ sudo npm install -g hexo 当Node.js和Git都安装好后就可以正式安装Hexo了
    输入管理员密码（Mac登录密码）即开始安装
    sudo: linux系统管理指令
    -g: 全局安装
    ① $ sudo npm install -g hexo-cli 若上面命令出错的话,执行这一行命令(若正常,忽略此命令)
    因为Hexo官网上的安装命令是$ npm install -g hexo-cli(window的命令)，
    Mac安装时不要忘记前面加上sudo，否则会因为权限问题报错
  3. 初始化Hexo本地文件夹,在mac终端命令行依次执行下面命令































    步骤 命令 解释
    ① $ cd work/Blog 终端cd到一个你选定的目录,如: 进入work文件夹下Blog文件夹中
    ② $ hexo init yassblog.xyz 这是hexo初始化命令,
    系统会自动初始化hexo文件夹. 文件夹名称任意(作者的是yassblog.xyz)
    ③ $ cd yassblog.xyz 终端cd到刚生成的hexo文件夹 (yassblog.xyz)中
    ④ $ npm install 安装npm
    ⑤ $ hexo s 开启hexo服务器

    此时, 浏览器中打开网址http://locsalhost:4000，能看到如下页面:

    到这里本地hexo环境基本设置好了,可以去关联github了
    另外:按照第3步可以在本地初始化多个Hexo文件夹,如图:
  4. 登录Github网站( https://github.com ),创建仓库(参考: github博客搭建一)

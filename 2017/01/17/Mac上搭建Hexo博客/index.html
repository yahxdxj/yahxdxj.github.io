Mac上搭建Hexo博客

发表于 2017-01-17

导语

为搭建Hexo博客,踩了无数的坑,终究是搭建成功了,高兴之余记录一下.

之前买过香港主机,用wp建站,试过之后每隔个1,2月会挂掉,咨询空间商告诉说主机IP受到大规模DDOS攻击,导致不能访问,问题往复几次,个人接受不了,放弃.遂转向Github,使用Github搭建博客也有好处,最直接的一点就是免费.话不多说,开始记录.

Mac上搭建基于GitHub的Hexo博客可分5步:

  1. 搭建github仓库及域名解析;
  2. Hexo关联Github;
  3. 安装主题theme;
  4. 绑定个人域名;
  5. 创建并发布博客文章;

一 搭建github仓库及域名解析

  1. 创建github账号(github官网),一定要验证邮箱(验证邮箱)
    (这是作者新注册的github账号 用户名:ceshiblog 密码:y**95**84 邮箱:134****978@qq.com )
  2. 登录github,右上角点+按钮,创建仓库New repository
  3. 进入Create a new repository页面,
    • ①输入Repository name,要注意格式:你的github用户名.github.io,比如:ceshiblog.github.io
    • ②输入Description描述,比如个人博客,对于描述随意填,是可以改的
    • ③选择public
    • ④点击 Create repository
  4. 进入仓库页面 (作者的仓库页面) ,点击setting
  5. 向下滚动,设置主题(Change theme),选择喜欢的主题,随意选个就行
    选择一个主题并确定
  6. 仓库建好了,博客网站也基本建好 (https://ceshiblog.github.io)

  7. 绑定个人域名(关于域名的购买,自行百度,其实非常简单)

    • 现在我们不想通过输入https://ceshiblog.github.io来访问个人博客网站,而是希望通过自己的域名(如:http://ceshiblog.xyz)来访问

    • 下面详细步骤

      • ① 进入仓库页面的设置(Settings)选项卡(作者的仓库页面 )
      • ② 往下滑动,找到域名卡片(Custom domain),输入购买的域名(ceshiblog.xyz是作者在阿里云买的域名,约¥1元/年),然后保存
      • ③ 保存成功的显示: Custom domain “ceshiblog.xyz” saved

      • ④ 到域名商那里去域名解析(域名在哪里买的就到哪里去解析,作者去的是阿里云) 

        名词解释:
    域名解析: 通过域名查找IP地址并访问空间的过程叫做域名解析
     CNAME: 如果将域名指向一个域名，实现与被指向域名相同的访问效果，需要增加CNAME记录
        @ : 将域名解析为example.com（不带www），填写@或者不填写；
      www : 将域名解析为www.example.com，填写www；

      • ⑤ 等待10分钟,就可以通过个人域名(http://www.ceshiblog.xyz 或http://ceshiblog.xyz)访问博客网站了

  8. 手动clone一份代码,因为后面或许要用到
    打开仓库页面(作者的仓库页面 ),
    点击按钮,点击download zip,将代码clone到本地电脑,保存即可.

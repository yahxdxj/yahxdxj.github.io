MacDown语法

发表于 2017-01-16

一 MacDown 和 作者

MacDown 由John Gruber发布,旨在提供一个易读写的编辑工具,它继承原始Markdown的语法规范 Markdown原始语法.

MacDown 能极快的将文本内容转成HTML,并展示出来.

MacDown 支持所有原始Markdown语法规范,而且支持的更多,各种流行的但非标准语法是可以开启/关闭的

二 基础语法

在我告诉你额外语法功能之前,我先向您介绍基本语法规范,如果您已知道基本语法,可直接跳到下面学习额外语法 Markdown preference pane.

2.1 如何换行

强制换行:在一行的最末敲两次空格加一次回车

2.2 加粗 and 斜体

加粗: 用双星号包裹**Strong** or __Strong__ (Command-B)
斜体: 用单星号包裹*Emphasize* or _Emphasize_[^emphasize] (Command-I)

2.7 引用

尖括号 > 被用来块引用.
从技术上讲,引用好几段内容,并不是每一行都需要以尖括号>开始,只要段落与段落之间没有空行就行
尽管看上去有点丑陋.

块引用可以嵌套.

多级嵌套.

Most markdown syntaxes work inside block quotes.

  • Lists
  • [Links][arbitrary_id]
  • Etc.

2.8 内联代码

Inline code is indicated by surrounding it with backticks:
`Inline code`

If your code has `backticks` that need to be displayed, you can use double backticks:
``Code with `backticks` `` (mind the spaces preceding the final set of backticks)

2.9 代码块

If you indent at least four spaces or one tab, I’ll display a code block.

print('This is a code block')
print('The block must be preceded by a blank line')
print('Then indent at least 4 spaces or 1 tab')
